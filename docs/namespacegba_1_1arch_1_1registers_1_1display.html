auto& gba::arch::registers::display::bg_controls = at<std::array<uint16_t, 4>>(0x0400'0008)
static

BG[0-3] layer control registers.

Bit Explanation
0-1 Priority.
2-3 Character base block.
4-5 Unused (must be 0).
6 Mosaic enabled.
7 Palette depth. 0: 16 colors, 1: 256 colors.
8-12 Screen base block (in units of 2 KB).
13 Display area overflow method.
See also
Overflow
14-15 Screen size.
See also
